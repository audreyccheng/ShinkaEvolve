<DESCRIPTION>
Implements a continuous gradient voting mechanism in the repair phase and a continuous confidence function.
1.  **Gradient Voting**: Instead of binary selection between local and peer values based on a hint, the algorithm now blends the best candidate with the hint if they are within a trust tolerance (20%). This allows the network state to converge towards the flow-balanced ideal over iterations, rather than just snapping to the nearest noisy measurement.
2.  **Continuous Calibration**: Replaces discrete confidence buckets with a continuous linear decay function (`1.0 - 2.5 * error`). This provides finer granularity for the evaluator and better rewards high-precision repairs while penalizing uncertainty proportionally.
</DESCRIPTION>

        def calibrate(val, peer_val, hint_val, status_conf, is_down):
            if is_down:
                return status_conf

            # Evidence Errors
            err_sym = 0.0
            if has_peer:
                denom = max(val, peer_val, 1.0)
                err_sym = abs(val - peer_val) / denom

            err_flow = None
            if hint_val is not None:
                denom = max(val, hint_val, 1.0)
                err_flow = abs(val - hint_val) / denom

            # Select Best Evidence
            # We trust the result if it matches EITHER the peer OR the flow hint.
            if err_flow is not None:
                best_err = min(err_sym, err_flow)
            else:
                best_err = err_sym

            # Continuous Confidence Calibration
            # Use a steeper decay to penalize errors quickly, ensuring high confidence implies high accuracy.
            # 1.0 at 0 error, ~0.8 at 5% error, ~0.0 at 40% error.
            conf = max(0.0, 1.0 - (2.5 * best_err))

            # Contradiction Penalty
            # If flow and symmetry strongly disagree, confidence drops because the situation is ambiguous.
            if err_flow is not None:
                conflict = abs(err_sym - err_flow)
                if conflict > 0.2 and conf > 0.5:
                    conf *= 0.8

            return conf * status_conf
