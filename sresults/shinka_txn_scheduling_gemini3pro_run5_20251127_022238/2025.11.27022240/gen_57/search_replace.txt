<DESCRIPTION>
Optimizes the scheduling algorithm by:
1.  Replacing static Gamma in Beam Search with a linear decay (1.8 -> 1.0) to transition from parallelism-focused construction to packing-focused construction.
2.  Adding a 'Tail Ruin' mode in ILS to specifically target the end of the schedule (critical path) for optimization.
3.  Enforcing LPT (Longest Processing Time) order when re-inserting transactions during the Recreate phase, which generally improves packing density.
4.  Replacing the stochastic Simulated Annealing local search with a deterministic 'Polish' (Gap Repair) pass that systematically attempts to move every transaction to its optimal position. This approach was found to be highly effective in prior high-scoring generations.

                current_cost = best_cost

        # B. Multi-Mode Ruin Strategy
        work_seq = list(current_schedule)
        removed_txns = []

        mode = random.random()

        if mode < 0.25:
            # Mode 1: Tail Ruin (Critical Path)
            # Targeting the end of the schedule where bottlenecks often accumulate
            bs = max(4, int(num_txns * 0.25))
            if len(work_seq) > bs:
                start = len(work_seq) - bs
                removed_txns = work_seq[start:]
                del work_seq[start:]

        elif mode < 0.60:
            # Mode 2: Single Large Block (Locality)
            bs = max(2, int(num_txns * 0.2))
            if len(work_seq) > bs:
                start = random.randint(0, len(work_seq) - bs)
                removed_txns = work_seq[start : start + bs]
                del work_seq[start : start + bs]

        else:
            # Mode 3: Two Disjoint Blocks (Global shuffle)
            total_rem = max(4, int(num_txns * 0.20))
            b1 = total_rem // 2
            b2 = total_rem - b1

            if len(work_seq) > b1:
                s1 = random.randint(0, len(work_seq) - b1)
                removed_txns.extend(work_seq[s1 : s1 + b1])
                del work_seq[s1 : s1 + b1]
            if len(work_seq) > b2:
                s2 = random.randint(0, len(work_seq) - b2)
                removed_txns.extend(work_seq[s2 : s2 + b2])
                del work_seq[s2 : s2 + b2]
