<DESCRIPTION>
Replace the simple pairwise swap refinement with a more powerful Large Neighborhood Search (LNS) using "Ruin and Recreate" strategy.
The LNS step selects the heaviest, lightest, and optionally a random third pack, unpacks them, and uses a randomized greedy heuristic to re-pack them.
This allows for 3-way exchanges and more complex reshuffling to escape local optima that pairwise swaps cannot.
Also optimize the main loop by pre-converting weights to Python lists and using fast list operations.
</DESCRIPTION>
