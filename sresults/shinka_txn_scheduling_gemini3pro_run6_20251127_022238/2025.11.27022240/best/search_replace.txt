<DESCRIPTION>
This edit implements three key optimization strategies:
1.  **Expanded Deterministic Anchors**: Increases the number of top LPT (Longest Processing Time) transactions selected deterministically in Beam Search from 3 to 4. This stabilizes the search by ensuring the most critical path candidates are always considered.
2.  **Deeper Lookahead**: Increases the lookahead target set from Top-3 to Top-4 LPT items, improving the algorithm's ability to foresee bottlenecks.
3.  **Aggressive ILS with Threshold Acceptance**:
    *   Increases the ILS cycles to 12.
    *   Increases Ruin size to 6-9 items to effectively disrupt complex local optima.
    *   Introduces Threshold Acceptance (0.5% tolerance) to allow the search to traverse "valleys" in the cost landscape rather than getting stuck in the first local minimum.
</DESCRIPTION>
