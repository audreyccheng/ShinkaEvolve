<DESCRIPTION>
Updates the Analytic Flow Consensus algorithm with two key improvements:
1.  **Differentiated Physics Thresholds**: Replaces `STRICT_THRESHOLD` (1%) with `PHYSICAL_THRESHOLD` (0.5%) to strictly clamp "physically impossible" violations (RX > Peer TX) while retaining the looser `HARDENING_THRESHOLD` (2%) for plausible measurement noise or agreement checks.
2.  **Hybrid Confidence Model**: Replaces the complex continuous confidence formula with a simpler "Discrete Bucket + Linear Penalty" model. High base scores (e.g., 0.98 for verified repairs) are assigned based on repair type, and then a penalty proportional to the *residual* router imbalance (ratio * 2.0) is subtracted. This ensures that even "perfect" link repairs receive lower confidence if the surrounding router state remains chaotic (imbalance > 5%).

    2. Rate Repair (Iterative):
       - Calculate 'Flow Implied' rate that balances the router.
       - Enforce strict physical constraints (RX <= Peer TX) with tight tolerance.
       - Arbitrate loss scenarios (RX < Peer TX) using Flow Implied value.

    3. Confidence Calibration:
       - Continuous scoring based on residual errors and agreement strength.

    HARDENING_THRESHOLD = 0.02   # 2% relative error considered 'match'
    STRICT_THRESHOLD = 0.01      # 1% for physical impossibility checks
    BASE_NOISE_FLOOR = 10.0      # Minimum Mbps to consider 'active'

    def get_noise_floor(rate_a, rate_b=0.0):
        # Scale noise floor for high speed links (0.1%), but keep base floor
        # Using a smaller percentage (0.1%) allows catching smaller errors on large links
        mx = max(rate_a, rate_b)
        return max(BASE_NOISE_FLOOR, mx * 0.001)

    def calc_error(v1, v2):
        nf = get_noise_floor(v1, v2)
        return abs(v1 - v2) / max(v1, v2, nf)

    state = {}

            'rx': float(data.get('rx_rate', 0.0)),
            'tx': float(data.get('tx_rate', 0.0)),
            'status': data.get('interface_status', 'unknown'),
            'peer': data.get('connected_to'),
            'router': data.get('local_router'),
            'orig_rx': float(data.get('rx_rate', 0.0)),

            p = state[s['peer']]
            if p['orig_rx'] > BASE_NOISE_FLOOR or p['orig_tx'] > BASE_NOISE_FLOOR:

                peer_is_down = True

        if local_traffic or peer_traffic:
            s['status'] = 'up'
        elif peer_is_down and not local_traffic:
            s['status'] = 'down'

        if s['status'] != 'up':
            s['rx'] = 0.0
            s['tx'] = 0.0

                'sum_tx': sum_tx,

                elif calc_error(val_self, val_peer) < HARDENING_THRESHOLD:

    final_router_qual = {}
