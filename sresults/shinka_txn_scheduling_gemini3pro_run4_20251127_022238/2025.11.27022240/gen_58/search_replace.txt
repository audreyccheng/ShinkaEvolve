<DESCRIPTION>
Implements a robust multi-stage "funnel" search strategy with diversity-driven greedy construction.
1. **Diverse Greedy**: Varies the 'Big Rock' threshold and length-weighting alpha per candidate to explore different structural packings (Recommendations #1).
2. **Warmup**: Applies a short LAHC run to all candidates before filtering, improving the quality of the Sprint pool (Recommendation #2).
3. **Enhanced LAHC**:
    - Adds a **Sampled Greedy Insert** operator to exploit best positions for randomly selected items (Recommendation #4).
    - Increases **Block Shift** size to [2, 8] to move larger dependency chains (Recommendation #3).
    - Adds a **Stagnation Kick** (shuffling a segment) in the Marathon phase to escape local optima (Recommendation #5).

def get_best_schedule(workload, num_seqs):

    Get optimal schedule using Adaptive Greedy and Multi-Stage LAHC Refinement.

    Algorithm:
    1.  **Txn Analysis**: Compute transaction operation counts.
    2.  **Adaptive Greedy Phase**:
        -   Generate `num_seqs` candidates.
        -   **Adaptive Pool**: Select "Big Rocks" (txns within 90% of max remaining length) + Randoms.
        -   **Weighted Selection**: Minimize `cost - (alpha * length)` (Packing heuristic).
        -   **Polish**: Adjacent Swap Descent on each greedy candidate.
    3.  **Multi-Candidate Sprint**:
        -   Select top 3 distinct candidates.
        -   Run short Late Acceptance Hill Climbing (LAHC) on each.
    4.  **Marathon Refinement**:
        -   Select winner of Sprint.
        -   Run extended LAHC with Block Moves.

        -   **Adaptive Construction**: Select "Big Rocks" + Randoms. Minimize `cost - (alpha * length)`.
        -   **Warmup Refinement**: Apply a short LAHC to every candidate to fix local greedy artifacts.
    3.  **Sprint Phase**:
        -   Select top 3 distinct candidates after Warmup.
        -   Run medium-length LAHC on each.
    4.  **Marathon Phase**:
        -   Select winner of Sprint.
        -   Run extended LAHC with **Sampled Greedy Insert**, **Deep Block Moves**, and **Stagnation Kick**.
