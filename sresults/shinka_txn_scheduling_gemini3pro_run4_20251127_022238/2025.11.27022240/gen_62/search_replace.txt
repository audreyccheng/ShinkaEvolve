<DESCRIPTION>
1.  **Adaptive LAHC History**: Modified `run_lahc` to dynamically set the history length based on the iteration budget (`budget // 20`). This ensures that short phases like "Warmup" and "Sprint" converge quickly (using a short history of 5-15) instead of random walking with a large empty buffer, while the "Marathon" phase retains a long history (up to 100) for deep exploration.
2.  **Greedy Diversity**: Randomized the `rock_limit` (the number of heavy transactions prioritized) between 2 and 8 during the greedy construction phase. This creates more structural diversity in the initial candidates than a fixed limit of 6.
3.  **Operator Tuning**: Adjusted local search operator probabilities to 45% Single Insert, 45% Block Insert, and 10% Swap (previously 50/40/10). Increasing the Block Insert weight helps move dependent transaction chains together.
4.  **Funnel Widening**: Increased the number of candidates promoted from the Warmup to the Sprint phase from 3 to 4, broadening the search during the intermediate optimization stage.

    # Helper: LAHC Optimization
    def run_lahc(schedule, start_cost, budget, use_kick=False):
        current_sched = list(schedule)
        current_cost = start_cost
        best_sched = list(schedule)
        best_cost = start_cost

        history_len = 50
        history = [start_cost] * history_len
        last_imp_idx = 0

        for k in range(budget):
            # Stagnation Kick (only enabled in Marathon phase)
            if use_kick and (k - last_imp_idx > 500):
                # Shuffle a random segment to escape local optima
                idx = random.randint(0, max(0, len(current_sched) - 15))
                seg_len = random.randint(10, 20)
                end = min(len(current_sched), idx + seg_len)
                segment = current_sched[idx:end]
                random.shuffle(segment)
                current_sched[idx:end] = segment

                # Reset history and costs
                current_cost = workload.get_opt_seq_cost(current_sched)
                history = [current_cost] * history_len
                last_imp_idx = k
                continue

            op_rand = random.random()
            neighbor = list(current_sched)

            # Operators: 50% Insert, 40% Block Insert (2-8 items), 10% Swap
            if op_rand < 0.50:
                # Single Insert
                idx1 = random.randint(0, len(neighbor) - 1)
                idx2 = random.randint(0, len(neighbor) - 1)
                if idx1 != idx2:
                    item = neighbor.pop(idx1)
                    neighbor.insert(idx2, item)
            elif op_rand < 0.90:
                # Block Insert
                bsize = random.randint(2, 8)
                if len(neighbor) > bsize:
                    start = random.randint(0, len(neighbor) - bsize)
                    block = neighbor[start : start + bsize]
                    del neighbor[start : start + bsize]
                    dest = random.randint(0, len(neighbor))
                    neighbor[dest : dest] = block
                else:
                    continue
            else:
                # Swap
                idx1, idx2 = random.sample(range(len(neighbor)), 2)
                neighbor[idx1], neighbor[idx2] = neighbor[idx2], neighbor[idx1]

            new_cost = workload.get_opt_seq_cost(neighbor)

            v = k % history_len

                current_cost = new_cost

                    best_cost = current_cost
                    best_sched = list(current_sched)
                    last_imp_idx = k
