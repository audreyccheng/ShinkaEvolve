<DESCRIPTION>
1.  **Conflict-Aware Heuristic**: I introduce a `conflict_score` for each transaction, counting how many other transactions it conflicts with. In the Greedy phase, I add a `beta` parameter (randomized) that weights this conflict score. This encourages placing "high-conflict" transactions early in the schedule, which is a known good heuristic for scheduling with dependencies (resolving bottlenecks early). Both `alpha` (length weight) and `beta` (conflict weight) decay over the construction process.
2.  **Intensification**: Since the program runs quickly, I increased the `MARATHON_ITERS` from 3500 to 5000.
3.  **Enhanced Mutation**: I increased the number of samples in the "Sampled Best-Fit" mutation from 3 to 6 to find better insertion points, as this is a high-quality (but expensive) operator.

def get_best_schedule(workload, num_seqs):
    """
    Get optimal schedule using Conflict-Aware Diverse Greedy and Intensified LAHC.

    Algorithm:
    1.  **Txn Analysis**: Pre-compute transaction lengths and conflict degrees (number of conflicting peers).
    2.  **Diverse Greedy Phase**:
        -   Generate `num_seqs` candidates.
        -   **Dual Decay Heuristic**: Weights for 'length' (alpha) and 'conflict_degree' (beta) decay over time.
        -   **Goal**: Place "Big Rocks" and "Hot Rocks" (high conflict) early.
        -   **Warmup**: Short LAHC.
    3.  **Sprint Phase**:
        -   Run LAHC on top 3 candidates.
    4.  **Marathon Phase**:
        -   Run extended LAHC (5000 iters) on best candidate.
        -   **Enhanced Mutation**: 'Sampled Best-Fit' checks more positions (6 samples).
    """

    # --- Pre-computation ---
    txn_lens = {}
    conflict_scores = {}

    # Parse for conflicts
    txn_r_sets = []
    txn_w_sets = []

    try:
        # Pass 1: Parse all
        for i in range(workload.num_txns):
            raw_txn = workload.txns[i]
            if isinstance(raw_txn, (list, tuple)):
                raw_txn = raw_txn[0]
            txn_str = str(raw_txn)

            # Ops for length
            all_ops = re.findall(r'([rw])-(\d+)', txn_str)
            txn_lens[i] = len(all_ops)

            # Sets for conflict detection
            r_set = set()
            w_set = set()
            for op_type, key in all_ops:
                if op_type == 'r':
                    r_set.add(key)
                else:
                    w_set.add(key)
            txn_r_sets.append(r_set)
            txn_w_sets.append(w_set)

        # Pass 2: Compute Conflict Degrees (O(N^2))
        for i in range(workload.num_txns):
            conflicts = 0
            w_i = txn_w_sets[i]
            rw_i = w_i.union(txn_r_sets[i])

            for j in range(workload.num_txns):
                if i == j: continue
                w_j = txn_w_sets[j]
                rw_j = w_j.union(txn_r_sets[j])

                # Conflict if: W_i intersects RW_j OR W_j intersects RW_i
                if not w_i.isdisjoint(rw_j) or not w_j.isdisjoint(rw_i):
                    conflicts += 1
            conflict_scores[i] = conflicts

            txn_lens[i] = 1
            conflict_scores[i] = 0
